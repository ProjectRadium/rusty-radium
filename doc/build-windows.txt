To build on Windows, follow these directions:

Install the required dependencies:
 - Git
 - Rust (https://static.rust-lang.org/dist/rust-1.8.0-x86_64-pc-windows-gnu.msi)

Run the following commands:
 - git clone https://github.com/tm2013/rusty-radium.git
 - cd path\to\rusty-radium
 - cargo build --release
 
That's it! Quick and painless.
